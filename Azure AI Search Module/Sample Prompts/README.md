## Usage Notes

- The Azure AI Search plugin must be configured in your workspace before using these prompts.
- Always include **"Azure AI Search"** in your prompt to invoke the plugin.
- Be specific in your queries — the plugin uses hybrid semantic and keyword search, so more specific prompts return more relevant results.
- Replace `{placeholders}` with your actual values.
