# Security Copilot Workshop

Comprehensive workshop materials for Microsoft Security Copilot in Microsoft Purview, providing prompt engineering best practices, automated investigation capabilities, and practical guides for insider risk management investigations.

## Overview

This repository contains curated content designed to help security analysts and administrators effectively leverage Security Copilot for:

- Automated insider risk alert investigations
- Intelligent alert prioritization and triage
- Rapid threat response and analysis
- Embedded security investigation workflows

## Contents

### Purview Module

Comprehensive guides for using Security Copilot in Microsoft Purview for Insider Risk Management investigations.
